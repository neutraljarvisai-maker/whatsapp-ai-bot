# VECTA OS - UI

This is the cinematic desktop interface for VECTA, a Batman-inspired AI operating system.

## Setup

1. Navigate to this directory:
   ```bash
   cd client/ui
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Design System

- **Primary Glow**: Cyan (#00D1FF)
- **Background**: Matte Black (#000000)
- **Warning**: Amber (#F59E0B)
- **Typography**: Inter (Sans) and Fira Code (Mono)

## Visual Principles

- Subtle glassmorphism
- 1px thin cyan outlines
- High contrast readability
- Grid-locked alignment
- Minimal, cinematic animations
